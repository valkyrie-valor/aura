Aura

Install APKs from Nexus or local storage

Download Aura v1.0.0

Requires Android 10+ (API 30)

Features

Nexus Repository Manager

Browse and install APKs directly from your Nexus 3 raw repository.

Local File Storage

Pick any APK from your device via the system file picker and install it instantly.

Material 3 Expressive Design

Dynamic color theming (Material You), adaptive layout for phones and tablets.

German & English Support

Fully localized in English and German with a per-app language override setting.

Release Notes

What's new in 1.0.0 Initial release
  • Browse and install APKs from Nexus Repository Manager 3 (raw format)
  • Install APKs from local file storage via system file picker
  • Recent picks list with Room persistence
  • Installed apps list with uninstall support
  • Streaming download with progress indicator
  • Download progress notification via foreground service
  • Settings: server URL, repository name, theme (system / light / dark), dynamic color toggle, language override
  • Material 3 Expressive UI with dynamic color (Material You)
  • German and English localization
  • NavigationSuiteScaffold with adaptive layout for phones and tablets
